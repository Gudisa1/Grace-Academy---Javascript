
# **Modifying Content and Styles in the DOM**

---

## **Step 1: Understanding Content vs Styles**

* **Content** → What is **inside the element**, like text, HTML, or attributes.
* **Styles** → How the element **looks on the page**, like color, size, font, background, borders.

Think of it like a **card**:

* Content → what’s written on the card.
* Styles → the **color, font, and decoration** of the card.

---

## **Step 2: Selecting the element first**

Before modifying, you must **select the element**.

```html
<h1 id="title">Hello World</h1>
<p class="text">This is a paragraph.</p>
```

```javascript
const title = document.getElementById("title");
const paragraph = document.querySelector(".text");
```

✅ Always select the element **before changing content or styles**.

---

## **Step 3: Modifying Content**

### **3a: Using `textContent`**

* Changes **text inside the element** (ignores HTML).

```javascript
title.textContent = "New Heading!";
```

✅ Result: `<h1>` now displays **New Heading!**

---

### **3b: Using `innerHTML`**

* Changes **HTML content inside the element**.

```javascript
paragraph.innerHTML = "This is <strong>bold</strong> text now!";
```

✅ Result: Paragraph shows **bold text**, because `innerHTML` can parse HTML tags.

**Tip:** Use `textContent` when you want **plain text**, `innerHTML` when you want **HTML inside**.

---

### **3c: Using `innerText`** (optional detail)

* Similar to `textContent`, but **respects CSS styling** like `display: none`.
* For most cases, `textContent` is recommended.

---

## **Step 4: Modifying Styles**

* Styles can be changed via the **`style` property**.

```javascript
title.style.color = "blue";          // text color
title.style.backgroundColor = "yellow"; // background
title.style.fontSize = "36px";       // font size
title.style.border = "2px solid red"; // border
```

✅ Result: `<h1>` becomes **blue text with yellow background, 36px size, and red border**.

---

### **Step 5: Adding / Removing Classes**

* Instead of changing **individual styles**, use **CSS classes**.

HTML:

```html
<p id="para">Paragraph with style.</p>
```

CSS:

```css
.highlight {
  color: white;
  background-color: green;
  font-weight: bold;
}
```

JavaScript:

```javascript
const para = document.getElementById("para");

// Add class
para.classList.add("highlight");

// Remove class
// para.classList.remove("highlight");

// Toggle class
// para.classList.toggle("highlight");
```

✅ Result: The paragraph **turns green with white bold text**.

**Tip:** Using classes keeps JavaScript simple and separates **content from style**.

---

## **Step 6: Changing Attributes**

* Attributes like `src`, `href`, or `title` can also be modified.

```html
<img id="myImage" src="old.jpg" alt="Old Image">
```

```javascript
const img = document.getElementById("myImage");
img.src = "new.jpg";          // change image source
img.alt = "New Image";        // change alt text
img.setAttribute("title", "Hover text"); // add title attribute
```

✅ Result: Image updates, and hovering shows **new title**.

---

## **Step 7: Combining Content and Style Changes**

You can **change text and style together** to make updates visible immediately.

```javascript
const box = document.createElement("div");
box.textContent = "I am a box!";
box.style.color = "white";
box.style.backgroundColor = "purple";
box.style.padding = "10px";
box.style.margin = "10px";
box.style.fontSize = "20px";
document.body.appendChild(box);
```

✅ Result: A **purple box with white text** appears on the page.

---

## **Step 8: Practical Example – Button to Modify Content and Styles**

HTML:

```html
<h2 id="heading">Original Heading</h2>
<p id="para">Original paragraph.</p>
<button id="changeBtn">Change Content & Style</button>
```

JavaScript:

```javascript
const heading = document.getElementById("heading");
const para = document.getElementById("para");
const button = document.getElementById("changeBtn");

button.addEventListener("click", () => {
  heading.textContent = "Updated Heading!";
  heading.style.color = "red";
  heading.style.fontSize = "40px";
  
  para.innerHTML = "This is <em>updated</em> paragraph text!";
  para.style.color = "blue";
  para.style.backgroundColor = "lightyellow";
});
```

✅ **Result:**

* Click the button → heading and paragraph **update content and styles instantly**.
* Students can **see exactly how DOM updates happen**.

---

## **Step 9: Key Takeaways**

1. **Select elements first** (`getElementById`, `querySelector`).
2. **Content**: `textContent` (plain text), `innerHTML` (HTML).
3. **Styles**: use `style` property or **CSS classes** (`classList.add/remove/toggle`).
4. **Attributes**: change with `.attributeName` or `setAttribute()`.
5. **Combine changes** for visual, interactive effects.
6. **Always test visually**—it makes it easy to understand what your code does.
